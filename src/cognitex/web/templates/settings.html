Setup Instructions

1. Install cognitex-sync on your local machine: 

# Download and install
curl -sSL {{ request.url_for('download_sync_installer') }} | bash

# Or manually with pip
pip install cognitex-sync

2. Configure with your server: 

cognitex-sync configure \
  --server {{ request.base_url }} \
  --api-key {{ sync_api_key or 'YOUR_API_KEY' }}

3. Sync your sessions: 

# One-time sync
cognitex-sync push

# Or run as background daemon
cognitex-sync daemon --interval 30
